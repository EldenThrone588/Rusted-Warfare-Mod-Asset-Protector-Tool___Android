# Security Policy

## **Supported Versions**

❌ Older Versions
✅ Latest Versions

## **Reporting a Issue/Vulnerability**

**If you find a security issue in RW Mod Protector:**

1. **Create a Github issue if and only it is important**
2. **Contact me directly** via:
   - Discord: [Join our server](https://discord.gg/WKYHkd53Y6)
   - Patreon: [Direct message](https://www.patreon.com/c/smallbutbigsizeofbug)

3. **Include details:**
   - Phone model(if the case was crashing..etc)
   - Android version
   - Tool version
   - Steps to reproduce
   - Description of the issue

## **What Will Happen Next**

- I'll acknowledge your issue report within No-work period.
- I'll investigate and confirm the issue.
- If its a valid issue, then I'll work on a immediate fix.
- You'll be credited (if you want) when the fix is released.

- **Note:** I dont held accountable for any tool feature abuse. As i may please, avoid abusive behavior for this tool. Thank you.


Thank you for helping to keep my RW Mod Protector Tool safe!
